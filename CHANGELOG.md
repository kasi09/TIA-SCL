# Changelog

## [0.4.0]

### Added
- **Symbol Outline & Breadcrumbs** - Document symbols in Outline panel showing block/variable hierarchy
- **Go to Definition** - Ctrl+Click on `#variables` and `"BlockName"` references to jump to declaration
- **Code Formatting** - Format Document (`Shift+Alt+F`) with proper indentation, keyword casing, operator spacing
- **Quick Fixes** - Auto-fix for missing VERSION, missing pragma, CASE without ELSE, naming conventions

## [0.3.0]

### Added
- Real-time linter with 11 diagnostic rules (5 errors, 5 warnings, 1 hint)

## [0.2.0]

### Added
- IntelliSense / Autocomplete for keywords, types, functions, variables
- Hover documentation for all keywords and system functions
- Signature help for system function calls

## [0.1.0]

### Added
- Syntax highlighting for SCL files
- 35+ code snippets for all block types and patterns
- Code folding, auto-indentation, bracket matching
